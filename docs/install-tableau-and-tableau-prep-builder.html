16.1 How to Install Tableau and Tableau Prep Builder

  • Tableau offers free one-year Tableau licenses to students, which includes access to entire eLearning platform.

  • Navigate to this link and click “Get Tableau for Free”. Then follow the registration instructions. After which you will receive an email with further information, where you will need to do the following:

    1. Download and install “Tableau Desktop” AND “Tableau Prep”.

    2. Open Tableau Desktop application and enter the product key given in the email to activate the license. Note that this will be valid for one year, but it can be renewed / applied for again.

    3. Create an account for eLearning and register with the given access code.

16.2 Next Assignment

  • Our first work with Tableau will be the DataCamp assignment “Introduction to Tableau”.

  • Even if it takes a while for Tableau to respond to your application, this assignment can be completed without Tableau installed on your computer.

  • As you will see, within DataCamp there is a version called Tableau Public, which would be sufficient for starting out.

16.3 Tableau eLearning

  • This is Tableau’s online teaching platform that contains tons of great resources in the form of interactive courses or learning paths (kinda like a plan of courses).

  • These would be great supplementary resources as we are working through Tableau or a great way to continue learning Tableau outside of our class.

  • I encourage you to browse around here to see what is available!
